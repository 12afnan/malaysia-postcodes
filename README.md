# Malaysia Postcodes

List of Malaysia Postcodes with City and State (JSON)

Postcodes, City and State are displayed in **JSON** in these following formats

```javascript
{
   "name": "Kelantan",
   "city": [
       {
        "name": "Pasir Mas",
        "postcode": [
            "17000",
            "17007",
            "17009",
            "17010",
            "17020",
            "17030",
            "17040",
            "17050",
            "17060",
            "17070"
        ]
       }
   ]
}
```
