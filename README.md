# Malaysia Postcodes

List of Malaysia Postcodes with City and State (JSON)

Postcodes are displayed in **JSON** in these following formats

```javascript
{
   "name": "Kelantan",
   "city": [
       {
        "name": "Pasir Mas",
        "postcode": ["17070"]
       }
   ]
}
```
